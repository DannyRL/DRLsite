    Projects

    Adaptive Vehicle Suspension System

    The controller pictured on the top left of this page was designed and built from scratch by myself as part of a personal project. I'm working on building an adjustable air suspension system for my car. The main advantages to this over a traditional spring and shock system is adjustable ride height and comfort. I'm implementing features such as allowing the user to manually adjust the height of the vehicle, a self-leveling system, and an adaptive ride.

    Whenever weight is added to a vehicle, it typically sags a bit as the springs in the suspension system get compressed. However in my system, the controller will sense this sag in height and raise the car back up. This creates a safer riding condition as all four wheels of the car can be balanced. Also as the car travels along the road, it will encounter bumps. The spring and shock system of traditional cars help to absorb some of the forces experienced by bumps in the road, but they are only reactive at best. My system will actively sense bumps and dips in the road and constantly adjust the suspension so that it will "fight back" against bumpy road conditions. This adds a significant level of comfort to the ride. This project is currently still in a prototype phase, but I hope to complete it early next year.
